# Covid 19 LineUp.js and UpSet.js Demos

This is a demo repository how [LineUp.js](https://lineup.js.org) and [UpSet.js](https://upset.js.org) can be used to explore Covid 19 data. 

as [Websites](#websites), via [Jupyter Notebooks](#jupyter-notebooks), and [R and R Shiny apps](#r-studio).

Jupyterlab: [![Open in Binder][binder]](https://mybinder.org/v2/gh/sgratzl/covid19_demos/master?urlpath=lab)
RStudio: [![Open in Binder][binder]](https://mybinder.org/v2/gh/sgratzl/covid19_demos/master?urlpath=rstudio)
